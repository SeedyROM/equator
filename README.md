Equational image generator JSON-API, written with Flask and Pillow 

### How to use locally? 
* `git clone https://github.com/SeedyROM/equator && cd equator`
* `pipenv install && pipenv shell`
* `./bin/develop.sh`
* `<url>/?eq=<equation_to_draw>&w=<width>&h=<height>&s=<scale>`
